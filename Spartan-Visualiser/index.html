This is a visualisation/simulation tool of the GoI-style programming language framework Sᴘᴀʀᴛᴀɴ, by Dan R. Ghica, Koko Muroya and Todd Waugh Ambridge.
Visualiser implemented by Todd Waugh Ambridge (based on previous work by Steven Cheung, using graph-viz-d3-js for generating diagrams and lo-js for parsing).
